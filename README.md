# No Drunk & Hangover

A mod for [My Winter Car](https://store.steampowered.com/app/4164420/My_Winter_Car/) that removes drunk and hangover effects from the player.

## Description

This mod continuously resets the player's drunk and hangover values to zero, effectively preventing any drunk or hangover effects from occurring in-game.

## Requirements

- [MSCLoader (for My Winter Car) Nexusmods](https://www.nexusmods.com/mywintercar/mods/3) Recommended
- [MSCLoader (for My Winter Car) Github](https://github.com/piotrulos/MSCModLoader)

## Installation

1. Make sure MSCLoader is installed.
2. Download the latest [release](https://github.com/LonoxX/MWC-NoDrunkHangover/releases)
3. Place the DLL file into the `Mods` folder of your My Winter Car installation directory.

## Building from Source

1. Clone this repository.
2. Open the solution in Visual Studio or your preferred IDE.
3. Ensure the game is installed at the default Steam path, or set the `GameReferencePath` property to point to your `mywintercar_Data\Managed` folder.
4. Build the project. The DLL will automatically be copied to the game's `Mods` folder.
